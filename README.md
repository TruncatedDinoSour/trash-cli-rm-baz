# Baz plugin: trash-cli-rm-baz

> Baz plugin for aliases for trash-cli

# Installation

- Using [baz plugin manager](https://ari-web.xyz/gh/baz):

```bash
$ baz install git 'https://ari-web.xyz/gh/trash-cli-rm-baz'
```

# Aliases

- `rls` -- List trash
- `rm` -- Put into trash
- `rmk` -- Restore trash
- `rrm` -- Clear trash
